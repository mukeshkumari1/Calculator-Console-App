
# Calculator Console App

## Overview

A Calculator console application is a program designed to perform basic operations such as add, subtract, multiplication division etc. and shows history of performed operations.

## Features

- Perform basic operations such as add, subtract, multiplication, division, modulo, factorial, root, exponential etc.
- Save and load operation history: Persistant storage of data between sessions

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or later installed.
- Your favorite Java IDE (optional).

### Build and Run

1. Clone the repository:

   ```bash
   git clone https://github.com/mukeshkumari1/Calculator-Console-App.git

2. Run the CalculatorConsoleApp.java file.

### Usage 
Once the application is running, you can perform  following operations:
- Addition
- Subtraction
- Multiplication
- Division
- Modulo
- Factorial
- Exponential
- Root
- Log
- Sine
- Cosine
- Percentage
- Conversion(feets to meters | meters to feets )
- Show History of operations
